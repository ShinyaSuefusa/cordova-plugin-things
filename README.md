# cordova-plugin-things
Cordova用AndroidThingsプラグインです。

現在のサポート：Android Things 1.0

注意：あまりメンテできていませんので悪しからず。

# 使い方
Cordova/Monacaにプラグイン追加してください。

細かな設定方法はいずれ記載します。。

cordova.plugins.things からアクセスできます。

# 対応機能
 - androidthings
 - gpio
 - i2c
 - uart
 - button
 - keypad
 - ssd1306
 - lcdpcf8574
 - rc522
